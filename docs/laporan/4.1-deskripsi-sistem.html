4.1 Deskripsi Sistem

SISKAOBE (Sistem Informasi Capaian Kurikulum Outcome Based Education) adalah sistem informasi berbasis web yang dikembangkan untuk mengelola dan memonitor pencapaian Capaian Pembelajaran Lulusan (CPL) mahasiswa berdasarkan pendekatan Outcome Based Education (OBE).

Gambaran Sistem yang Diusulkan

Sistem ini dirancang untuk membantu program studi dalam melakukan pemetaan CPL terhadap mata kuliah, mengelola CPMK (Capaian Pembelajaran Mata Kuliah), serta menghitung dan memantau tingkat pencapaian CPL mahasiswa berdasarkan nilai yang diperoleh.

Pengguna Sistem

No Pengguna Peran
1 Admin Mengelola data master (user, prodi, mahasiswa, tahun akademik)
2 Tim Kurikulum Mengelola CPL, mata kuliah, CPMK, dan pemetaan CPL-MK
3 Dosen Menginput nilai mahasiswa per CPMK
4 Kaprodi Melihat dashboard dan laporan pencapaian CPL
5 Wadir 1 Melihat dashboard dan laporan pencapaian CPL seluruh prodi

Input, Proses, dan Output

Komponen Keterangan
Input Data CPL, data mata kuliah, data CPMK, pemetaan CPL-MK, nilai mahasiswa per CPMK
Proses Pemetaan CPL ke mata kuliah, perhitungan bobot CPMK, kalkulasi pencapaian CPL berdasarkan nilai mahasiswa
Output Dashboard pencapaian CPL, laporan pencapaian CPL per mahasiswa, laporan pencapaian CPL per angkatan, rekap nilai CPMK
